# Movimiento-con-teclas-en-JS
Este proyecto muestra una luna que puede moverse por la pantalla mediante las teclas de dirección del teclado. Además, genera varios cráteres dentro de la luna con posiciones aleatorias para simular una apariencia más realista.

Características principales

Movimiento mediante las teclas de flecha: arriba, abajo, izquierda y derecha.

La luna incluye sombras dinámicas que cambian según la dirección del movimiento.

Generación automática de varios cráteres distribuidos aleatoriamente dentro de la luna.

Interfaz minimalista y centrada en la experiencia visual.

Tecnologías utilizadas

HTML5

CSS3

JavaScript (DOM y eventos de teclado)

Estructura del proyecto

index.html: Contiene la estructura principal del documento.

style.css: Incluye los estilos aplicados a la luna y el fondo de la página (si se maneja mediante un archivo externo).

script.js: Implementa el movimiento de la luna y la generación de cráteres.
